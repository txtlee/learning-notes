# 打家劫舍2

## 题目

## 解题思路
打家劫舍的动态规划
以及，环形，无非就是打劫第一家或不打劫第一家

## 代码
class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) == 1:
            return nums[0]
        def robs(nums):
            prev1 = 0
            prev2 = 0
            for num in nums:
                cur = max(prev1, prev2 + num)
                prev1, prev2 = cur, prev1
            return prev1
        a = robs(nums[1:])
        b = robs(nums[: len(nums) - 1])
        return max(a, b)
