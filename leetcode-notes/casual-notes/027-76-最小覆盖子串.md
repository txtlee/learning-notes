# 最小覆盖子串

## 题目
给定两个字符串 s 和 t，长度分别是 m 和 n，返回 s 中的 最短窗口 子串，使得该子串包含 t 中的每一个字符（包括重复字符）。如果没有这样的子串，返回空字符串 ""。

测试用例保证答案唯一。

 

示例 1：

输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
示例 2：

输入：s = "a", t = "a"
输出："a"
解释：整个字符串 s 是最小覆盖子串。
示例 3:

输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。

## 解题思路
滑动窗口 + 计数字典：双哈希 + 计数器(need_cnt)

另外：
1. from collections import defaulutdict
   dic = defaultdict(int)  # key不存在时不报错，相当于常规声明的执行 dict.get(ch, 0) + 1 再更行才不会报错
2. ans = float('inf') 正无穷
   ans = float('-inf') 负无穷

## 代码
class Solution(object):
    def minWindow(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: str
        核心思路：need_cnt
        """
        if len(t) > len(s):
            return ""
        from collections import defaultdict

        need = defaultdict(int)
        for ch in t:
            need[ch] += 1
        need_cnt = len(need)
        window = defaultdict(int)

        ans_len = float('inf')
        ans_left = 0
        left = 0
        for right, ch in enumerate(s):
            window[ch] += 1
            if ch in need and window[ch] == need[ch]:
                need_cnt -= 1
            while need_cnt == 0:
                cur_len = right - left + 1
                if cur_len < ans_len:
                    ans_len = cur_len
                    ans_left = left
                l_ch = s[left]
                window[l_ch] -= 1
                if l_ch in need and window[l_ch] < need[l_ch]:
                    need_cnt += 1
                left += 1
        print(ans_len)
        return "" if ans_len == float('inf') else s[ans_left: ans_left + ans_len]
