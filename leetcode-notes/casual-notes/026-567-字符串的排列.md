# 字符串的排列

## 题目
给你两个字符串 s1 和 s2 ，写一个函数来判断 s2 是否包含 s1 的 排列。如果是，返回 true ；否则，返回 false 。

换句话说，s1 的排列之一是 s2 的 子串 。

 

示例 1：

输入：s1 = "ab" s2 = "eidbaooo"
输出：true
解释：s2 包含 s1 的排列之一 ("ba").
示例 2：

输入：s1= "ab" s2 = "eidboaoo"
输出：false

## 解题思路
滑动窗口 + 计数
计数常用技巧：
count = [0] * 26
idx = ord(s[i]) - ord('a')

## 代码
```python
class Solution(object):
    def checkInclusion(self, s1, s2):
        """
        :type s1: str
        :type s2: str
        :rtype: bool
        """
        if len(s1) > len(s2):
            return False
        count = [0] * 26
        for i in range(len(s1)):
            idx = ord(s1[i]) - ord('a')
            count[idx] += 1
        for i in range(len(s1)):
            idx = ord(s2[i]) - ord('a')
            count[idx] -= 1
        if not any(count):
            return True
        for i in range(len(s1), len(s2)):
            idx_left = ord(s2[i - len(s1)]) - ord('a')
            idx_right = ord(s2[i]) - ord('a')
            count[idx_left] += 1
            count[idx_right] -= 1
            if not any(count):
                return True
        return False
```
