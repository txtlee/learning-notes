# 重排链表

## 题目
给定一个单链表 L 的头节点 head ，单链表 L 表示为：

L0 → L1 → … → Ln - 1 → Ln
请将其重新排列后变为：

L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

## 解题思路
三步法：链表中点 -> 链表反转 -> 链表合并
做链表的题时要时刻记得上述三步中的任何一步都可能实现解题

空间换时间法：将链表同步到list中

## 代码
class Solution(object):
    def reorderList(self, head):
        """
        :type head: Optional[ListNode]
        :rtype: None Do not return anything, modify head in-place instead.
        """
        # 链表中点 + 链表反转 + 链表合并
        # 链表中点
        if not head.next:
            return head
        def getMiddle(head):
            slow = head
            fast = head.next
            while fast and fast.next:
                slow = slow.next
                fast = fast.next.next
            node2 = slow.next
            slow.next = None
            return head, node2
        head1, head2 = getMiddle(head)
            # 链表反转
        def reverse(node):
            dummy = ListNode(0)
            ptr = node
            while ptr:
                ptr_nxt = ptr.next
                ptr.next = dummy.next
                dummy.next = ptr
                ptr = ptr_nxt
            return dummy.next
        new_head2 = reverse(head2)

        # 合并链表
        def merge(head1, head2):
            dummy = ListNode(0)
            node1 = head1
            node2 = head2
            while node1 and node2:
                node1_nxt = node1.next
                node2_nxt = node2.next
                dummy.next = node1
                node1.next = node2
                dummy = node2
                node1 = node1_nxt
                node2 = node2_nxt
            if node1:
                dummy.next = node1
            if node2:
                dummy.next = node2
            return head1
        
        res = merge(head1, new_head2)
        return res
