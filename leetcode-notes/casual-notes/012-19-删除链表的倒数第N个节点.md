# 删除链表的倒数第N个节点

## 题目
给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例1:
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

示例2:
输入：head = [1], n = 1
输出：[]

## 解题思路
    快慢指针，也叫双指针
    删除倒数第k个数，快指针先走k步，然后快慢指针同时走，当快指针走到尾节点的next即null节点时，慢节点处于待删除节点位置；
    需要注意到是，删除的如果是第一个节点，属于边界问题，需要特别处理下

## 代码
class Solution(object):
    def removeNthFromEnd(self, head, n):
        """
        :type head: Optional[ListNode]
        :type n: int
        :rtype: Optional[ListNode]
        """
        slow = head
        fast = head
        pre = head
        for i in range(n):
            fast = fast.next
        while fast:
            pre = slow
            slow = slow.next
            fast = fast.next
        if pre == slow:
            return pre.next
        pre.next = slow.next

        return head
