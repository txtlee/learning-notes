# 接雨水

## 题目
给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。


示例 1：


输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
示例 2：

输入：height = [4,2,0,3,2,5]
输出：9

## 解题思路
双指针数组？NONONO，这叫 **动态规划**
left[i] 记录当前柱子 i 左侧最高
right[i] 记录当前柱子 i 右侧最高
柱子 i 能容纳的水量取决于其左右最高的柱子及其本身高度

进阶：双指针
当前柱子容纳的水取决与 leftMax 和 rightMax 中的较小值；
leftMax < rightMax时，计算左指针处水量并移动左指针；
leftMax > rightMax时，计算右指针处水量并移动右指针

## 代码

class Solution(object):
    def trap(self, height):
        """
        :type height: List[int]
        :rtype: int
        """
        n = len(height)
        left = [None] * n
        right = [None] * n
        left_max = 0
        right_max = 0
        for i in range(n):
            if height[i] > left_max:
                left_max = height[i]
            left[i] = left_max
        for i in range(n - 1, -1, -1):
            if height[i] > right_max:
                right_max = height[i]
            right[i] = right_max
        res = 0
        for i in range(n):
            if height[i] < left[i] and height[i] < right[i]:
                res += min(left[i], right[i]) - height[i]
        
        return res
