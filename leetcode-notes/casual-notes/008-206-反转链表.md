# 反转链表

## 题目
给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

示例1：
输入：head = [1,2]
输出：[2,1]

示例2：
输入：head = []
输出：[]

## 解题思路
三种思路：迭代法、递归法、头插法
1. 迭代法
   先存下一个节点；
   当前节点指向前一个；
   prev 前进一步；
   curr 前进一步；
   循环结束时 prev 就是新头；

    pre = None
    cur = head
    while cur:
        nxt = cur.next
        cur.next = pre
        pre = cur
        cur = nxt

    return pre

2. 递归法
    先递归反转后面的,再把当前节点接上去；
    终止条件:空链表或只剩一个节点,直接返回；

    def rec(head):
        if not head or not head.next:
            return head
        res = rec(head.next)

        head.next.next = head
        head.next = None     
        return res

    return rec(head)

3. 头插法
    哨兵节点,简化头插逻辑；

    dummy = ListNode(0)
    cur = head
    while cur:
        nxt = cur.next
        cur.next = dummy.next
        dummy.next = cur
        cur = nxt
    
    return dummy.next
